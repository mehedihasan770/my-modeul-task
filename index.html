CSS Layout and Styling Properties

How to Use Margin?

In CSS, margin is used to add space outside an element. It separates elements and helps with layout spacing. You can use margin: 20px; for equal space on all sides, or margin: 10px 20px; for top-bottom and left-right. To center a block element, use margin: 0 auto;. Margin can be written in inline style, internal CSS, or external CSS. It only affects the space outside the element, not the content inside.

How to Use Padding?

In CSS, padding adds space inside an element, between its content and border. It makes content look neat and gives breathing room. Use padding: 20px; for equal space on all sides, or padding: 10px 20px; for top-bottom and left-right. You can also set four sides individually. Padding can be written in inline, internal, or external CSS. Unlike margin, padding affects the space inside the element.

How to Use Font?

In CSS, the font controls the style of the text. You can change its typeface, size, weight, and appearance. For example, font-family sets the font style like Arial or Times New Roman. font-size changes how big the text looks. font-weight makes text bold or light. You write these inside CSS to make your webpage text look attractive and readable.

How to Use Flex box?

Flexbox is a CSS layout tool that helps arrange items in a container easily. It allows you to align, space, and order elements in a flexible way, either horizontally or vertically. By setting display: flex; on a parent container, its children become flexible boxes. You can control direction with flex-direction, alignment with justify-content and align-items. Flexbox makes building responsive and clean layouts simple.

How to Use Grid?

CSS Grid is a powerful layout system for creating two-dimensional grids on a webpage. It lets you divide a container into rows and columns, placing items precisely where you want. By setting display: grid; on a container, you can define grid columns and rows using grid-template-columns and grid-template-rows. Grid makes complex layouts easier and more flexible than older methods.

How to Use background Img?

In CSS, you can set an image as the background of an element using background-image. The syntax is: background-image: url("image.jpg"); You can control how the image behaves using other properties like background-size (to resize), background-repeat (to stop repeating), and background-position (to set its place). Background images are useful for making sections of a website more attractive.
